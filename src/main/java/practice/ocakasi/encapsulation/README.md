## *JavaBeans*
### Rules for *JavaBeans* naming conventions
| **Rule**                                                    | **Example**                                 |
|-------------------------------------------------------------|---------------------------------------------|
| *Properties* are `private`                                  | `private int numEggs;`                      |
| *Getter* methods begin with `is` if the property is boolean | `public boolean isHappy() {return happy; }` |
